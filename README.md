Directions on using GIT,gitHub and webstorm.

*****GIT***** is a VCS (Version Control System) used for code management and GitHub is website that offers online repository services.
You can use git withuot GitHub but you cannot use GitHub without git. Webstorm is an Integrated Development Environment(IDE) used for coding in JavaScrript, HTML, CSS.
Webstorm lets you managed Git projects hosted on GitHub directly from the IDE:***clone***, repositories, etc.
Step by Step on how to use Git, GitHub & Webstorm.

1. Install Webstorm [https://www.jetbrains.com/webstorm/download/#section=windows], Git[https://git-scm.com/download/win] and create a github account on Github.com.
2. In Webstorm, from the main menu choose Git / clone. The Git manu is available when you have set up a Git ***repository*** for your project until then is VCS.
3. In the Get from version control dialog choose ***GitHub*** on the left.
4. In the directory field enter the path to the folder where the local git reposirory will be created.
5. Click clone. You can add a ***remote*** GitHub repository for a project you are developeing locally.
